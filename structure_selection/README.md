#Structure Selection

For the filtering of the excel file (from the results): 
1. first filter for the x-ray diffraction as a method
2. look at the resolution and only focus on A<2.5
3. check the coverage of the file
4. remove the protein complexes
5. then selected the structure that didnt have any mutations, with the highest possible coverage and best resolution. 
6. the one that I ended up choosing was the 2IMT and this is because of the resoultion being reasonably good, it covered all the known motifs that are important for its function (only missing one residue) and it had no mutations.
